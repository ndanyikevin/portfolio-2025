## 🧑‍💻 Getting Started

To run the project locally:

```bash
# Clone the repo
git clone https://github.com/ndanyikevin/portfolio-2025.git
cd portfolio-2025

# Install dependencies
npm install

# Run the development server
npm run dev

# Visit the site
http://localhost:3000
````
